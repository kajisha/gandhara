# Gandhara

In Rails4, timestamps columns are null: true by default.
`Gandhara` adds null: false to timestamps.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gandhara'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gandhara
